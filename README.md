## Overview
Developed a web-based legal agreement automation platform to generate and review legal documents, reducing manual effort and improving document processing efficiency.

## Features
- Generate customized legal agreements
- Review and manage multiple agreement types
- Automated expiry notifications and tracking
- Digital agreement signing integration

## Implementation
- Developed backend services using Flask framework
- Integrated DocuSign API for digital signing workflow
- Implemented automated notifications using AWS SES and Cron jobs
- Designed database management using AWS S3 and DynamoDB
- Deployed the application on AWS cloud infrastructure

## Tech Stack
Python | Flask | AWS SES | AWS S3 | DynamoDB | DocuSign API | Cron Jobs

## Impact
Reduced manual agreement generation and review effort by 80% through automation and improved legal workflow efficiency.
